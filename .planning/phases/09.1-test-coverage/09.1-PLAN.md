# Phase 9.1 — Comprehensive Test Coverage

## Goal

Add integration tests, property tests, UI automation tests, and cross-platform sandbox escape tests to every crate in the Kay workspace that currently lacks coverage. After this phase every crate has at least one test file; CI runs on macOS, Linux, and Windows; and a coverage gate enforces the requirement going forward.

## Success Criteria

- **TEST-01** — Every crate in the workspace has at least one test file under `tests/` or `src/` (integration or unit). Verified by `coverage-gate.sh`.
- **TEST-02** — All public APIs of `forge_*` crates are exercised by at least one integration test. Verified by `cargo test -p <crate>` green on all 18 crates.
- **TEST-03** — `kay-sandbox-{macos,linux,windows}` each have a subprocess escape test suite that executes real `std::process::Command` calls, gated by `#[cfg(target_os)]`. All pass on their respective OS in CI.
- **TEST-04** — `kay-tauri` has `tests/commands.rs` exercising `start_session`, `stop_session`, and `get_session_status` via `tauri::test::mock_builder()`.
- **TEST-05** — WebDriverIO UI smoke suite (`e2e/smoke.ts`) with ≥5 test cases passes via `tauri-driver` on macOS CI.
- **TEST-06** — CI matrix runs on `ubuntu-latest`, `macos-latest`, `windows-latest` for all Rust tests. Sandbox tests are OS-gated.
- **TEST-07** — `scripts/coverage-gate.sh` is a required CI step that fails the build if any gap-list crate has zero test files.

### W-7 Task 7.3 — Coverage gate script

**Action:**
```
Create scripts/coverage-gate.sh:
#!/bin/bash
set -euo pipefail

GAP_LIST=(
  "forge_app" "forge_config" "forge_display" "forge_domain" "forge_embed"
  "forge_fs" "forge_infra" "forge_json_repair" "forge_main" "forge_spinner"
  "forge_markdown_stream" "forge_repo" "forge_services" "forge_snaps"
  "forge_stream" "forge_template" "forge_tracker" "forge_walker"
  "forge_api" "forge_ci" "forge_test_kit" "forge_tool_macros"
  "kay-sandbox-macos" "kay-sandbox-linux" "kay-sandbox-windows"
  "kay-tauri" "kay-tui"
)

FAILED=0
for crate in "${GAP_LIST[@]}"; do
  # Find crate directory
  dir=$(find crates -maxdepth 1 -name "$crate" -type d 2>/dev/null | head -1)
  if [ -z "$dir" ]; then
    echo "WARN: crate directory not found for $crate"
    continue
  fi
  # Check for test files
  test_count=$(find "$dir/tests" -name "*.rs" 2>/dev/null | wc -l)
  src_test=$(grep -rl "#\[test\]" "$dir/src" 2>/dev/null | wc -l)
  total=$((test_count + src_test))
  if [ "$total" -eq 0 ]; then
    echo "FAIL: $crate has zero test files"
    FAILED=1
  else
    echo "OK:   $crate ($total test file(s))"
  fi
done

if [ "$FAILED" -eq 1 ]; then
  echo ""
  echo "Coverage gate FAILED: one or more gap-list crates have no tests."
  exit 1
fi
echo ""
echo "Coverage gate PASSED: all gap-list crates have at least one test file."

chmod +x scripts/coverage-gate.sh
```

**Verify:** `./scripts/coverage-gate.sh` — PASSED after all waves complete.
